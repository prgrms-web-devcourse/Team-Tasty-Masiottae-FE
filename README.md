## 실행 방법

프로젝트 clone 후 서버를 실행합니다. 
기본 포트는 [http://localhost:3000](http://localhost:3000) 3000으로, 서버 실행 후 해당 주소로 접속하시면 됩니다. 

### 개발 서버 실행 
```bash
npm run dev
# or
yarn dev
```
### 프로덕션 서버 실행
```bash
npx next build
npx next start
```
